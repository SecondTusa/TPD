INSTALLATION GUIDE:

MacOS:

Warning:

This program can change MacOS defaults (like system language, hotkeys etc.) 

Open it only on your own risk!

Recomended Mac Version: 10.13.6

1) Open downloaded .dmg file (This is for Intel. Silicon Version will be later)
2) Move .app file to desktop
3) Open .app file
4) If you get error like "This app cant be trusted" follow next steps
5) After opening close error and go to system settings
6) Click Privacy and Security
7) Click Open Anyway
8) Then app will open and program will start
